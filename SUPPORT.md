# Support

Use GitHub issues for reproducible bugs, board-support requests, feature requests, and hardware validation evidence.

Before opening an issue, please include:

- PulseWetProbe version
- Board model and Arduino core version
- Example/sketch name
- Wiring topology and resistor/reference details
- Compile logs or CSV output where relevant

PulseWetProbe is a trend-oriented sensing library. It is not a certified RWIS sensor, EC meter, salinity meter, or freezing-point detector.
